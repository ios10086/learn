    # 生命周期

    # 小程序注册生命周期

    属性 类型 默认值 必填 说明 最低版本
    onLaunch (opens new window) function 否 生命周期回调——监听小程序初始化。
    onShow (opens new window) function 否 生命周期回调——监听小程序启动或切前台。
    onHide (opens new window) function 否 生命周期回调——监听小程序切后台。
    onError (opens new window) function 否 错误监听函数。
    onPageNotFound (opens new window) function 否 页面不存在监听函数。 1.9.90 (opens new window)
    onUnhandledRejection (opens new window) function 否 未处理的 Promise 拒绝事件监听函数。 2.10.0 (opens new window)
    onThemeChange (opens new window) function 否 监听系统主题变化 2.11.0 (opens new window)
    其他 any 否 开发者可以添加任意的函数或数据变量到 Object 参数中，用 this 可以访问

    # 小程序页面生命周期

    属性 类型 默认值 必填 说明
    data (opens new window) Object 页面的初始数据
    options Object 页面的组件选项，同 Component 构造器 (opens new window) 中的 options ，需要基础库版本 2.10.1 (opens new window)
    behaviors (opens new window) String Array 类似于mixins和traits的组件间代码复用机制，参见 behaviors (opens new window)，需要基础库版本 2.9.2 (opens new window)
    onLoad (opens new window) function 生命周期回调—监听页面加载
    onShow (opens new window) function 生命周期回调—监听页面显示
    onReady (opens new window) function 生命周期回调—监听页面初次渲染完成
    onHide (opens new window) function 生命周期回调—监听页面隐藏
    onUnload (opens new window) function 生命周期回调—监听页面卸载
    onPullDownRefresh (opens new window) function 监听用户下拉动作
    onReachBottom (opens new window) function 页面上拉触底事件的处理函数
    onShareAppMessage (opens new window) function 用户点击右上角转发
    onShareTimeline (opens new window) function 用户点击右上角转发到朋友圈
    onAddToFavorites (opens new window) function 用户点击右上角收藏
    onPageScroll (opens new window) function 页面滚动触发事件的处理函数
    onResize (opens new window) function 页面尺寸改变时触发，详见 响应显示区域变化 (opens new window)
    onTabItemTap (opens new window) function 当前是 tab 页时，点击 tab 时触发
    onSaveExitState (opens new window) function 页面销毁前保留状态回调
    其他 any 开发者可以添加任意的函数或数据到 Object参数中，在页面的函数中用 this 可以访问。这部分属性会在页面实例创建时进行一次深拷贝。
